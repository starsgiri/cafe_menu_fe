# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by creating an issue in the GitHub repository.

## Supported Versions

This is a static website project. Security updates will be applied to the latest version.

## Security Best Practices

This project follows these security practices:
- No sensitive data stored in code
- No backend or database connections
- Static HTML/CSS only
- External resources loaded via HTTPS
