# Cafe Menu

A simple, elegant cafe menu website showcasing coffee and desserts with a vintage aesthetic.

## Features

- Responsive design
- Clean and organized menu layout
- Coffee and desserts sections
- Vintage styling with custom fonts and colors

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout

## Deployment

This is a static website that can be hosted on GitHub Pages.

### Steps to Deploy:

1. Push your code to GitHub
2. Go to your repository settings
3. Navigate to "Pages" section
4. Select the branch (main/master) and root folder
5. Click Save
6. Your site will be live at: `https://starsgiri.github.io/cafe_menu_fe/`

## Local Development

Simply open `index.html` in your web browser to view the menu locally.
